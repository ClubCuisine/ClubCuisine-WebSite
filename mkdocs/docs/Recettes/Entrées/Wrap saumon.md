## Instructions

### Etape 1

Etaler le boursin sur tout le wrap.

### Etape 2

Mettre les tranches de saumon fumé sur la longueur (environs sur tout le wrap pour que ca se roule bien).

### Etape 3

Rouler puis couper. c'est déjà pret !!!
